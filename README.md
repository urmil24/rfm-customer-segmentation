# Customer Segmentation (RFM Analysis)

## Problem Statement
Businesses struggle to identify valuable customers and optimize marketing strategies. This project uses RFM analysis to segment customers based on behavior.

## Tools Used
- Python (Pandas, Matplotlib)
- Power BI
- Excel

## Steps Performed
- Data Cleaning
- RFM Calculation (Recency, Frequency, Monetary)
- Customer Segmentation
- Data Visualization

## Key Insights
- Best customers contribute highest revenue
- At-risk customers need retention strategies
- Age group 26–35 spends the most
